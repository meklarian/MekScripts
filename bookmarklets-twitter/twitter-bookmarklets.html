@paul_shinn's twitter bookmarklets/favelets

To use these bookmarklets, you should right-click and add the bookmarks of the links below to your favorites.

If your browser has a 'favorite links' toolbar, place the bookmark there so that you can use it with a single click.

In some browsers, you may also be able to drag the link and drop it directly on the toolbar.

Once you have added these bookmarklets, you may click on them to activate them while viewing a twitter page (yours or anyone else). Users of older browsers will need to select them from their favorites menu instead.

Alternate (and improved) versions of some of these bookmarklets are available at @cutlerish's page here: http://infraware.ca/twoolset, including a Greasemonkey script for Firefox users and a Twitter Line-Break repair tool to fix multi-line tweets.
Also: A greasemonkey script to award you 1-Ups for every 100 stars on Twitter/Brizzly/Favstar.fm!

Add a Tools Palette to Twitter and Brizzly

Last Updated on July 25th, 2010

This bookmarklet adds handy links into the page to toggle @ replies and hashtags, and other useful things.

ToolsPalette v3

In twitter, these links will appear before the header of tweets. In brizzly, they will appear at the bottom of the left-hand menu list, with the heading 'Toggles'.

Twitter Features

  • Toggle @Mentions and #Hashtags
  • Disable Map Hovercard Pins
  • Filter Location Checkins (4square, Gowalla, Loopt)
  • Filter Personal Streams (formspring, blip.fm, ustream, last.fm, twitterfeed, rhythmbox, newzform, rss2twitter, auto-tweeting, chatter.com, sharethis, alltop)
  • Filter Game-Related Tweets (mafia wars, farmville, words/chess with friends, xbox network updates, raptr)
  • Inline Insert Youtube/TwitVid Videos and Image Thumbnails (twitpic, yfrog, brizzly, img.ly)
  • Filter out keywords of your choice. Also displays a Bookmarklet link so you can save and reuse your filters later.

Brizzly Features

  • Toggle @Mentions and #Hashtags

Toggle @ Replies/Mentions on Twitter and Brizzly

This bookmarklet can toggle back and forth between showing and hiding @ replies.

Works on both Twitter and Brizzly pages!

Toggle # Hashtags on Twitter and Brizzly

This bookmarklet can toggle back and forth between showing and hiding # hashtags.

Works on both Twitter and Brizzly pages!

Disable Location Links on Twitter

This bookmarklet will disable all location reference links on Twitter pages. The links will still be visible, but no hovercards or any other actions will occur when the links are clicked.

Disable Locations

Insert Picture/Video Previews on Twitter and Favstar.Fm

This bookmarklet will insert preview images for reference links that point to Twitpic, yFrog, Brizzly Pictures, and img.ly. In addition, it will expand YouTube and TwitVid links into embedded videos.

Previews v4
